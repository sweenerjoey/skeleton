About Skeleton.CSS

Inspired by Dave Gamache's framework of the same name, Skeleton.CSS uses a selection of intuitive utility classes to provide basic functionality for CSS grids and flexboxes, and establishes a basic spacing scale. No visual characteristics are applied through Skeleton.CSS, allowing users to build their own type scales, buttons styles, or entire design systems on top of the basic framework.

CSS Grid

My biggest critique of Gamache's Skeleton is that the 12-column grid isn't built using actual CSS grid functionality. My Skeleton.CSS addresses this by creating a mobile-first CSS grid that starts at 4 columns and expands to 12 columns as screen size increases. Along the way, utility classes that are device-specific allow you to place the direct children of a div with the .grid class onto the grid.

The utility classes follow a simple naming syntax, detailed in the next section. Most of the utility classes are aliased under various names, allowing you to use values like "thirds" and "quarters" as well as targeting specific gridlines.
